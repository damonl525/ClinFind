# 更新日志 (CHANGELOG)

## [v3.0.0] - 2026-01-31

### 🚀 全新架构重构
- **前端**：Vue 3 + TypeScript + Element Plus 现代化界面
- **后端**：Python FastAPI 高性能异步服务
- **架构**：前后端分离，Electron 桌面应用打包

### ✨ 新增功能
- **AI 智能搜索**：集成多种 AI 服务商（DeepSeek、OpenAI、智谱GLM、阿里通义、心流iflow等）
- **智能关键词扩展**：AI 自动分析搜索词并扩展同义词、相关词、专业术语
- **搜索范围管理**：支持多文件夹索引，可视化管理搜索范围
- **增量索引**：智能检测文件变化，只处理新增/修改的文件
- **实时索引状态**：可视化显示索引进度、文件数量、最后更新时间
- **逻辑搜索**：支持 AND/OR 组合搜索语法
- **供应商配置独立保存**：每个 AI 供应商的 API Key 和模型配置独立保存，切换时自动恢复

### 🎨 界面优化
- 现代化卡片式搜索结果展示
- 文件类型图标与颜色区分
- 搜索结果高亮显示匹配内容
- 响应式布局适配
- 平滑的动画过渡效果

### 🔧 技术改进
- 全文检索引擎优化，支持 PDF、Word、Excel、PPT 等格式
- 异步索引处理，不阻塞 UI
- 国际化支持（中文/英文）
- 完善的错误处理和日志记录

## [v2.1] - 2025-01-17

### 🎯 重大改进
- **搜索精度大幅提升**：解决了拼音缩写误匹配导致的搜索结果不准确问题
- **分层评分系统**：实现了精确匹配、部分匹配、语义匹配、拼音匹配、模糊匹配的分层权重控制

### ✨ 新增功能
- **智能意外匹配检测**：自动识别并降低拼音缩写与英文缩写的意外匹配得分
- **位置敏感性优化**：优先考虑词首匹配和完整词边界匹配
- **字符重叠分析**：增强了字符级别的匹配精度控制

### 🐛 问题修复
- 修复搜索"编盲"时"基于风险的监查RBM应用及实现"得分过高(90分→54分)的问题
- 修复拼音缩写'bm'与英文缩写'RBM'意外高分匹配的问题
- 优化了短字符串模糊匹配的准确性

### 📊 搜索结果优化
- 包含关键词的文档平均排名从5.2提升至3.0
- 不包含关键词的文档平均排名从4.8降至8.0
- 搜索结果相关性显著提升

### 🔧 技术改进
- 无硬编码规则，完全基于算法逻辑实现
- 向后兼容，保持原有API接口不变
- 自适应匹配类型识别机制
- 性能优化的分层处理，避免不必要计算
